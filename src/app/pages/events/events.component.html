Register Events

The following section details the requirements for all events to become ranked.

To have your event recognized as a ranked event by eSports Universal, your event must be posted in the #events-list channel with all of the following:

  • Day(s) of the week of the event
  • Calendar Date of the event(s)
  • Start time
  • Timezone

After two successful events have been hosted within one month, the third event will be the first officially ranked event under eSports Universal.

A minimum of two events must be hosted within a four-week time frame to maintain ranked status.

Host ranking status will reset if no events are held within 3 weeks.

Ranking Time Period per month:

  • First Ranking = 1st - 15th
  • Second Ranking Week = 16th - end of Month
